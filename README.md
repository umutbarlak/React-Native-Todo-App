<h1>React Native Todo App</h1>

Bu proje, React Native kullanılarak geliştirilen bir Todo uygulamasıdır. Uygulama, AsyncStorage kullanarak verileri kalıcı olarak saklar. Kullanıcılar todo ekleyebilir, silebilir ve güncelleyebilir. Ayrıca, onboarding ekranı yalnızca ilk girişte gösterilir ve istenirse sıfırlanarak yeniden görüntülenebilir.

<h2>Kullanılan Teknolojiler</h2>

- React Native: Mobil uygulama geliştirme.

- AsyncStorage: Kalıcı veri saklama.

- React Navigation: Sayfa yönlendirme.

- Lottie Animations: Animasyonlar.

- FlatList: Listeleme bileşeni.

- Iconsax: İkonlar.

<h2>Projenin bize sunduğu özellikler</h2>

- Todo Ekleme: Kullanıcılar yeni todo ekleyebilir.

- Todo Silme: Eklenen todoları silebilir.

- Todo Güncelleme: Mevcut todoları düzenleyebilir.

- AsyncStorage Kullanımı: Todolar cihazda kalıcı olarak saklanır.

- Onboarding Yönetimi: Kullanıcı uygulamaya ilk kez girdiğinde onboarding ekranı gösterilir. Daha sonra bir daha gösterilmez, ancak sıfırlanarak yeniden başlatılabilir.

- FlatList Kullanımı: Todolar liste şeklinde gösterilir.
